# Visuals

## Files

### results_visual.html
Open in any browser to see the complete visual summary:
- Coupling constants comparison (computed vs observed)
- Prime number chain (163 → 137 → 59 → 13 → 7 → 3)
- Boson mass comparison
- Lepton mass table
- Precession hierarchy (every particle's hidden vs emerged fraction)
- Cosmological parameters
- CP violation table
- The complete derivation chain

### two_cone_rotation.jsx
Adobe After Effects script that generates the key visualization:
two perpendicular vectors with 90-degree phase delay producing
two counter-rotating cones. This is the physical mechanism from
which everything follows.

To use: Open After Effects → File → Scripts → Run Script File →
select two_cone_rotation.jsx
